## Que es Fortris

Fortris es una plataforma que permite a usuarios de criptomonedas:

1. Formalizar legalmente su herencia con un testamento notarial digital
2. Crear una wallet inteligente basada en ERC-4337
3. Designar múltiples herederos con distribución de porcentajes
4. Garantizar que los herederos puedan reclamar sin necesidad de ETH (transacciones gasless)

## Stack Técnico

- **Frontend:** React/TypeScript + Paskeys
- **Backend:** Node.js + Validación PKI
- **Smart Contracts:** Solidity + ERC-4337
- **Cadenas:** Ethereum, Polygon, Arbitrum
- **Oráculo:** A definir (Chainlink o custom)
- **Paymaster:** Para gasless
